Numbers, Strings and Dates

More basic programming concepts.

Variables and Values

Javascript has three essential primitive data types:

  • numbers
  • strings
  • Booleans

While these are primitives, Javascript also provides a class of methods and properties for each of these types, which is a somewhat unique feature within Javascript.

Numbers

Unlike most programming languages, Javascript does not distinguish between integers and decimals - a number is a number. Numbers inherently interact with the mathematical operators:

  • plus
  • minus
  • multiply
  • divide
  • modulus (remainder)
  • exponent

The number object has methods such as toFixed( ) and Javascript also includes a Math class with many useful methods properties and constants.

Strings

Javascript strings are zero or more characters wrapped in single or double quotes. There is no character type - only strings with a length of one. Use the backslash \ escape character before inserting special characters into strings such as quotes and backslashes.

Strings can be concatenated using the plus operator. The Javascript String object has many useful methods and properties, such as .length. In addition, Javascript has built-in regular expression functions for very complex string management.
