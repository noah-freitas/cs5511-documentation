Numbers, Strings and Dates

More basic programming concepts.

Variables and Values

Javascript has three essential primitive data types:

  • numbers
  • strings
  • Booleans

While these are primitives (their abilities are built directly into Javascript) Javascript also provides a class of methods and properties for each of these types.

Numbers

Unlike most programming languages, Javascript does not distinguish between integers and decimals - a number is a number. Numbers inherently interact with the mathematical operators:

  • plus
  • minus
  • multiply
  • divide
  • modulus (remainder)
  • exponent

The number object has methods such as toFixed( ), which converts a number into a String, and Javascript also includes a Math class with many useful methods properties and constants for using trigonometry, algebra, rounding and randomization to your programming toolkit.

Strings

Javascript strings are defined as zero or more characters wrapped in single or double quotes. There is no character type - they are strings with a length of one. Use the backslash \ escape character before inserting special characters into strings such as quotes and backslashes.

Strings can be concatenated using the plus operator. The Javascript String object has many useful methods and properties, such as .length. In addition, Javascript has a Regular Expression object for more complex string operations.

Dates

In Javascript, values to represent date and time use the Date object. The Date() object has many methods for managing variables that hold date and time information.

In this script we create a dynamic Day Of The Week display in just few simple lines of Javascript, we use the Date() object and a "static" objectthat we define. 

var thePresent = new Date();
var display = document.getElementById("day");
var days = {0:"Sunday",1:"Monday",2:"Tuesday",3:"Wednesday",4:"Thursday",5:"Friday",6:"Saturday"};
var day = days[thePresent.getDay()];
display.innerHTML = "Today is " + day;

Take Aways

  • Programming typically involves managing information of different types of data
  • Data types often have their own inherent capabilities that need to be expressed programmatically
  • Data types and classes simplify the process of working with different types of information.
