More basic programming concepts.

Statements and Program Flow

In the previous lesson we learned that a STATEMENT is a 'line' of code that performs a task and is executed typically in sequence. In the rotate page example we worked on in the 8/21/2012 class we used statements that defined and assigned variables (these are sometimes called expression statements.) Javascript and other programming languages also have special statements that allow us to apply logic to our programming.

These special programming statements can not only change the condition of a variable, but also can change the normal, sequential flow of the program.

In this Rotate Page example we refine the logic of the page rotation function by getting it to stop after one revolution. Note in the code that we do this with an IF statement - a statement that 'tests' the situation (has the page element fully rotated yet) and acts accordingly. This new feature allows us to use LOGIC on the page - to teach the rotate function to stop itself at a certain point by giving it a CONDITION (in the parentheses following the IF).

We use a Javascript timer (setInterval()) and a recursive call to the rotate() function to interate, or repeat, the 1 degree rotation over time.

Javascript 'Logical' Statements

 

IF
Conditional - tests for a specific condition using the logical operators and directs program flow to either of one or two code blocks. False conditions include incorrect comparative expressions as well as the values false, null, undefined, the string "", the number 0 and the 'not a number' NaN.
if (degree !== 361) {
    rotate();
}

var y = (degree !== 361) ? rotate() : null;
Ternary
A similar technique to IF statements. Based on the truth of the test expression, a variable takes a different value or code path. It is good practice to have functions return values as we'll learn soon
        
var y = (degree !== 361) ? rotate() : null;
SWITCH
Performs a handy multi-path function based on evaluating an expression to match with a set of possible values defined by the case clause.
switch (day) {
    case "Sunday":
        sleep();
        break;
    case "Monday":
        work();
        break;
    case "Tuesday":
        play();
        break;
}
WHILE
A simple loop based on the 'truth' of the conditional expression.
while (i < 5) {
    x = x + (i*.26);
    i++; // an increment operator - add 1 to i
}
FOR
A more complex looping statement with the increment function defined right with the conditional expression. It also has a special version that increments through a set of objects. 
for (var i = 0; i < 5; i++) {
    x = x + "And a ..." + i;
}
 
for (x in person) {
    if (person.hasOwnProperty(x)) {
        console.log(person[x]);
    }
}
DO
Similar to while, but always executes it 'truth' block at least once and tests the truth expression after each execution.
do { 
    x++;
} 
while (x < 10);

Take Aways

  • Programmers add logic to programs to control program flow using special CONDITIONAL and ITERATIVE statements
  • There may be more than one way to program a specific situation. Some ways may be better than others and tradeoffs need to be evaluated.
