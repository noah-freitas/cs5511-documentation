JavaScript Fundamentals —Programming a Webpage

Learn how to incorporate JavaScript code on your webpages.

Introduction

JavaScript is a multi-paradigm scripting language that is prototype-based, dynamic, weakly-typed and has first-class functions and supports object-oriented, imperative, and functional programming styles.

Javascript uses sytax from C and naming conventions from Java. Javascript is a registered trademark of Oracle Corporation.paraphrased from WikiPedia

What does Javascript do?

  • Like HTML and CSS, Javascript is intepreted and executed by web browsers.
  • Javascript is a scripting language used to create dynamic behaviors between web content and users.
  • For the purposes of this class, we'll always assume unless stated otherwise, that we are referring to Javascript running within a web browser. However, Javascript as a programming language may be implemented in a variety of ways on different platforms and is becoming increasingly popular for web server scripting (node.js) and embedded programming.

Adding Javascript to a webpage

  • A script can be placed within a SCRIPT tag in your HTML file, within either the HEAD or the BODY element.
    • <script type="text/javascript">
 alert("Hello, World");
</script>
  • A script can be placed in a separate .js file which is referenced within the SRC attribute of a SCRIPT tag.
    • <script src="hello-world.js" type="text/javascript"></script>
  • A statement or short script can be placed in an event-handler attribute of a page element. 
    • <input type="button" onclick=alert("Hello, World"); />

Javascript 'Context' on a webpage

As you can see above, Javascript scripts can be loaded onto a webpage in many different ways. Once a script is loaded onto the page, it has scope of all CURRENT elements of the page and of the variables, functions and resources of the other scripts on the webpage.

However, a script does not have scope to page or script resources loaded onto the page subsequent to it. Wepages load line by line from the top down.

Some programming techniques load all scripts at the bottom of the page, others load their scripts wired to page events such as onLoad that fire after all page elements are loaded.

In web browsers, each SCRIPT tag delivers a compilation unit that is compiled and immediately executed...Javascript throws them all together in a common global namespace.Douglas Crockford, Javacript: The Good Parts

Terminology

Take Aways

  • Javascript, like HTML and CSS, is a feature of web browsers
  • Javascript enables web designers to add a 'behavior' dimension to webpages
  • Javascript is an accessible and easy way for web designers to learn programming
