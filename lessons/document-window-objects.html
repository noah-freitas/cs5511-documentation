Resources of the the web browser environment

What is the DOM?

We've already learned that virtually everything in Javascript is an Object, and that an Object in Javascript is a collection of properties..

When Javascript is included on a webpage loaded into a web browser, two special Objects are created at the global level - the document( )Object is constructed to represent the web page document and its contents, and the window( ) Object is constructed to represent the web browser environment.

The Document Object

The Document Object is constructed by the browser when the page loads and can be accessed by refering to it as document.

There are many properties (and methods) of the document object itself, as well as some of the child objects of document. In fact we've already seen some of these, to access, manipulate and create elements dynamically in the web document, as well as information related to the document, such as the source domain, cookies, character set and web protocol used to serve the document.

The Document Object contains several properties and methods to access references to page elements. Typically these return an array of references to elements that match the method call. Remember that the getElementById( ) will return a single object, not an array, as the Id value should be unique to the document.

In these two lines of Javascript, we reference the anchors property using dot notation on the document object, and then use dot notation on the array that is returned to find the number of links on a document.

var allLinks = document.anchors;
var linksCount = allLinks.length;

The Window Object

The web browser also constructs a Window( ) Object for each document that is loaded in a new window (or tab), in fact the Window Object contains a reference to the Document Object in it's window.document property.

We've already used methods of the Window( ) property in our projects, such as alert( ).

The Window object also contains the "Timer Event" methods. The setInterval() method repeats a function at a set time interval, while setTimeout() executes a function once after a delay.

In addition, the WIndow Object contains properties with information such as the screen height and width, pixel depth, history, and current URL.

Terminology

DOM
Document Object Model - the Object created to represent the document loaded into a browser window. Includes properties and methods to traverse and interact with the hierarchal set of nodes (elements) of a web document and it's attributes.
Window Object
An object created when a browser opens a new window or tab, containing information and methods pertaining to the browser's context, viewport size, timer, and other elements.

Take Aways

  • Web browsers construct Javascript objects automatically for the windows and documents that viewers load
  • Javascript is particularly useful in 'programming the DOM'
