What is the DOM?

We've already learned that virtually everything in Javascript is an Object, and that an Object in Javascript is a collection of properties..

When Javascript is included on a webpage loaded into a web browser, two special Objects are created at the global level - the document( )Object is constructed to represent the web page document and its contents, and the window( ) Object is constructed to represent the web browser environment.

The Document Object

The Document Object is constructed by the browser when the page loads and can be accessed by refering to it as document. . There are many properties (and methods) of the document object itself, as well as some of the child objects of document. In fact we've already seen some of these, to access, manipulate and create elements dynamically in the web document, as well as information related to the document, such as the source domain, cookies, character set and web protocol used to serve the document.

var jsClass = new Array();
jsClass[0] = "Steve";
jsClass[1] = "Shannon";
jsClass[2] = "Alex";

The Window Object

The web browser constructs a Window( ) Object for each document that is loaded in a new window (or tab), in fact the Window Object contains a reference to the Document Object in it's window.document property. We've already used methods of the Window( ) property in our projects, such as alert( ), setInternal( ) and setTimeout( ). In addition, the WIndow Object contains properties with information such as the screen height and width, pixel depth, history, and current URL.

jsClass = ["Steve","Shannon","Alex"];

Javascript Array Properties and Methods

The Array has several properties, such as length, and methods such as pop(), push(), slice() and splice() that are easy to use and make programming a with arrays a snap.

Multidimensional Arrays

Javascript does not have multidimensional arrays, in other words an array cannot have more than one index, as in a matrix. However an array can have an array as a member. It isn't possible to declare an array as x[0][0], it is possible to create such a construct programmatically with a FOR loop.

Associative Arrays

Javascript does not have associative arrays in the strictest sense, the standard array syntax does not support strings as indices. You can use syntax that appears to create an associative array, but you are really creating an object. Here's an interesting article that explains how associative arrays in Javascript are really objects. Another perspective, from Quirksmode, is that Objects in Javascript are associative arrays.

Terminology

Array
A variable with a set of values
Index
The position of a particular value within an array
Length
A property containing the CURRENT number of members of an array.
Multi-dimensional Array
An Array contained or containing an Array
Associative Array
An Array with named indices. Note the different syntax of curly braces and colons.
Splice
A method to remove members of an array
Slice
A method to copy an array or some of its members
Push
A method to add a member to the last index of an array
Pop
A method to remove the last member of an array
