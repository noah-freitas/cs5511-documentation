JQuery

A Javascript library for web designers.

Introduction

jQuery is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. jQuery is designed to change the way that you write JavaScript. jquery.org

JQuery includes the CORE library as well as JQuery UI and JQuery UI Mobile and other projects designed to make Javascript more accessible for developers.

Very Brief History

JQuery was originally designed to help developers overcome inconsistencies in the DOM implementation among different browsers. It became popular as an alternative to the basic Javascript syntax and DOM model.

JQuery vs. JavaScript

JQuery is not part of the core Javascript capabilities of web browsers and therefore the library needs to be loaded as a script using a script tag. Many developers choose to use a CDN hosted minified version of JQuery to improve pageload times through caching.

JQuery is essentially implemented as a Javascript function. The jquery( ) function takes as it's argument a string representing a 'selector' - almost identical to a CSS selector. The jquery( ) function has hundreds of useful methods for managing, manipulating and animating DOM objects and other elements.

JQuery and core Javascripts statements can co-exist within the same module or function, jquery( ) is nothing more than a Javascript function that extends the core capabilities of Javascript.

JQuery's methods typically act on the selector without the need to iterate through a collection (array).The syntax of JQuery is more expressive, similar to CSS.

JQuery function calls are typically made using the $ alias for simsimplicity.

JQuery Basics

//load the jquery library from google
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>

Accessing the collection of paragraphs in the document with a class of alarm and changing the text color to red. notice how the elements are selected, iterated and updated in one simple function call.

The function call includes several elements:

  • the JQuery function call - using the $ alias
  • the selector passed as an argument
  • the method (or action)
  • one or more potential arguments passed to the jquery method.
$("p.alarm").css("color","red");

 

Take Aways

  • JQuery is an ubiquitous library that simplifies Javascript programming and makes it more accessible to web designers
  • JQuery is implemented as a Javascript function and so can co-exist with Javascript statements in a program, sharing variables and namespaces.
  • JQuery includes a CORE library and also value added libraries for user interface and mobile development.
