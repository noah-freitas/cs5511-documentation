JavaScript Arrays

Collections of Objects

What is a Javascript Array?

An array in computer programming is a special type of variable that stores a collection of similar things, a variable than can hold more than one value at a time. As we've already seen in Javascript, all variables are objects of one type or another and the Javascript Array object is simply one that is particulary useful for collecting and managing sets of information. In working with collections or lists of information, arrays are often used with loops as an efficient means of programming.

Javascript Arrays are a little different that many other languages in that their members can actually be of different types. In addition, some languages require that arrays have predetermined lengths, whereas Javascript has no such restrictions.

var jsClass = new Array();
jsClass[0] = "Steve";
jsClass[1] = "Shannon";
jsClass[2] = "Alex";

Javascript Array Literals

An Array can be assigned a value to each index, or can be assigned a literal which contains a set of members, the literal would be enclosed in square brackets and each member separated by a comma.

jsClass = ["Steve","Shannon","Alex"];

Javascript Array Properties and Methods

The Array has several properties, such as length, and methods such as pop(), push(), slice() and splice() that are easy to use and make programming a with arrays a snap.

Multidimensional Arrays

Javascript does not have multidimensional arrays, in other words an array cannot have more than one index, as in a matrix. However an array can have an array as a member. It isn't possible to declare an array as x[0][0], it is possible to create such a construct programmatically with a FOR loop.

Associative Arrays

Terminology

Array
A variable with a set of values
Index
The position of a particular value within an array
Length
A property containing the CURRENT number of members of an array.
Multi-dimensional Array
An Array contained or containing an Array
Associative Array
An Array with named indices. Note the different syntax of curly braces and colons.
Splice
A method to remove members of an array
Slice
A method to copy an array or some of its members
Push
A method to add a member to the last index of an array
Pop
A method to remove the last member of an array

Take Aways

  • Use arrays for variables of collections
  • Javascript arrays can contain complex variables and objects
