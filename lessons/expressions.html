The Basic Building Blocks of a Programming Language

Computer languages 'abstract' real world information into data that can be manipulated. These terms and concepts will be familiar to people who have used programming languages such as C, Java, Basic, PHP, etc.

Terminology

Statement
A line of Javascript code typically ending with a semicolon that performs a function and is executed in line sequence. Specialized statements such as conditionals and iterators can contain inner blocks of statements.
Expression
A piece of Javascript code that produces a value. An expression can be a literal, a name (variable) and/or can contain operators on or between literals and variables.
Literal
An explicit expression of a value. Numbers, strings, objects, arrays and even functions can be expressed as literals in Javascript.
Variable
A named expression or value. In Javascript there are six basic types of values, or things that can be named inherently: numbers, strings, booleans, objects, functions and undefined. The keyterm VAR is used to create a new variable.

Adding Javascript to a webpage

  • A script can be placed within a SCRIPT tag in your HTML file, within either the HEAD or the BODY element.
    • <script type="text/javascript">
 alert("Hello, World");
</script>
  • A script can be placed in a separate .js file which is referenced within the SRC attribute of a SCRIPT tag.
    • <script src="hello-world.js" type="text/javascript"></script>
  • A statement or short script can be placed in an event-handler attribute of a page element. 
    • <input type="button" onclick=alert("Hello, World"); />

Javascript 'Context' on a webpage

As you can see above, Javascript scripts can be loaded onto a webpage in many different ways. Once a script is loaded onto the page, it has scope of all CURRENT elements of the page and of the variables, functions and resources of the other scripts on the webpage.

However, a script does not have scope to page or script resources loaded onto the page subsequent to it. Wepages load line by line from the top down.

Some programming techniques load all scripts at the bottom of the page, others load their scripts wired to page events such as onLoad that fire after all page elements are loaded.
