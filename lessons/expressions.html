The Basic Building Blocks of a Programming Language

Computer languages 'abstract' real world information into data that can be manipulated. These terms and concepts will be familiar to people who have used programming languages such as C, Java, Basic, PHP, etc.

Statement
A line of Javascript code typically ending with a semicolon that performs a function and is executed in line sequence.
A command to the browser - w3schools.com
var x;
x = 18;
var y = (x<21) ? "N" : "Y" // ternary assignment
Statements
Expression
A piece of Javascript code that produces a value. An expression can be a literal, a name (variable) and/or can contain operators on or between literals and variables.
x
8
"N"
Expressions
Literal
An explicit expression of a value. Numbers, strings, objects, arrays and even functions can be expressed as literals in Javascript.
(100 + 4) - 25 // numeric literal
"Javascript" // string literal
{reviewer:"George V",points:5}  // object literal
Literals
Variable
A named expression or value. The keyterm VAR is used to create a new variable. In Javascript there are six basic types of values, or things that can be named inherently: numbers, strings, booleans, objects, functions and undefined. Undefined is what makes Javascript a 'weakly-typed' programming language - types aren't required for defining variables. 
var x;
var y:string;
var z:Date;
Declaration statements
Value
The inherent value of an expression, typically a number, string or object. Javascript has built in values such as true, false, NaN (Not a Number), null, and undefined.
x = 21;
y = x-5;
var z = y * 12;
Assignment statements
Operator
Changes the value of expressions. There are mathematical, logical, assignment, comparison and other types of operators in Javascript.
x = 21; //assignment operator
y = x - 5; //mathematical MINUS operator
y = !y;  // logical NOT operator
Operators

Take Aways

  • Programmers express behaviors abstractly through expressions and statements
  • Javascript is similar to most programming languages, but has some interesting idiosyncrocies.

Assigning webpage elements as Javascript variables

We will learn more about what this means, but for now, it is convenient for us to just understand that this function: document.getElementbyId("") can be used to access an handle to a page element (with the appropriate id attribute) and assign it to a variable.
