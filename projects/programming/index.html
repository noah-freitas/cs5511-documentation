Project 1: Programming a HTML5 Grid with Javascript

What you will learn in this project

  • Javascript programming techniques
    • the FOR..IN statement
    • the IF statement
    • the DO..WHILE statement
    • the SWITCH statement
  • A quick introduction to DOM navigation and manipulation
    • getElementById( )
    • getElementsByClassName( )
    • getElementsByTagName( )
    • setAttribute( )
  • A quick overview of HTML5 grid layouts using Twitter Bootstrap
  1. Create a basic HTML5 webpage document 
    <!DOCTYPE HTML>

  2. Download the Twitter Bootstrap library. Link the bootstrap stylesheet in the head of your document and the Bootstrap JQuery and Javascript library in script tags before the ending of your document. Create a new Javascript file named programming.js and link that to your page with a script tag below bootstrap.js. This is where we will do our programming. 
    <link href=bootstrap/css/bootstrap.css rel=stylesheet>
</head>

<script type="text/javascript" src="bootstrap/js/bootstrap.js"></script>
<script type="text/javascript" src="programming.js"></script>
</body>
  3. Twitter Bootstrap has a built-in HTML/CSS grid system which is very easy to use. We are going to create a simple grid as a Javascript programming playground. You can use classes that are defined by Bootstrap such as 'row' and 'span(number)'. First create a div with id of grid and then rows with spans that add up to no more than 12. Feel free to make your own grid of regular or irregular numbers of columns. Create at least a few rows to work with - you can copy and paste the HTML from here.
