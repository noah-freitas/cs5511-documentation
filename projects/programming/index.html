The Grid

  1. First, create a basic HTML5 webpage document 
    <!DOCTYPE HTML>

<body>
</body>
</html>
  2. Download the Twitter Bootstrap library. Link the bootstrap stylesheet in the head of your document and jQuery (using Google's CDN) and the Bootstrap Javascript library in script tags before the ending of your document. Create a new Javascript file named programming.js and link that to your page with a script tag below bootstrap.js. This is where we will do our programming. 
    <link href=bootstrap/css/bootstrap.css rel=stylesheet>
</head>
...
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.0/jquery.min.js"></script>
<script src="bootstrap/js/bootstrap.js"></script>
<script src="programming.js"></script>
</body>
  3. Twitter Bootstrap has a built-in HTML/CSS grid system which is very easy to use. We are going to create a simple grid as a Javascript programming playground. You can use classes that are defined by Bootstrap such as 'row' and 'span(number)'. First create a div with id of grid and then rows with spans that add up to no more than 12. For this project it's OK to use the class "span1" for your blocks. Create at least a few rows to work with - you can copy and paste the HTML from here. 
    4. <div id="grid">
<div class="row">
<div class="span1">
</div>
<div class="span1">
</div>
<div class="span1">
</div>
</div>
...
</div>
  4. Now we can start having some fun with Javascript! First we will use a few functions that are build into web browsers for 'traversing' web documents. We'll learn more about the Document Object Model in a few weeks. For now, it's important for you just to know that these functions return object references to the page elements that meet the criteria we ask for. 
    5. // access the grid element - returns a single object
var grid = document.getElementById("grid");
// access the grid rows and cells elements - each returns a set of object
var spans = grid.getElementsByClassName("span1");
var rows = grid.getElementsByClassName("row");
  5. First, let's add content to each of the grid cells. Assigning a value to the innerHTML property of a webpage element updates its content. The value would typically be a string and can contain content and markup. 
    6. // the FOR statement allows us to iterate over each element of an array or DOM collection.
for (var ii = 0, len = spans.length; ii < len; ii++) {
    spans[ii].innerHTML = "
    " + ii + "
    ";
}
  6. Next we'll use a DO ... WHILE loop statement to change the properties of each of the paragraphs to align them properly. We can create a new set using getElementsByTagName(). The WHILE loop statements have slightly diferrent syntax than the FOR loop. NOTE that this statement also requires you to take on alot of responsibility. It is easy to create an INFINITE LOOP and freeze your browser session and possibly your operating system. Don't forget to increment the counter!
    7. var content = grid.getElementsByTagName("p"),
    x = 0,
    len = content.length;
    
do {
	content[x].style.width = "20px";
	content[x].style.margin = "10px auto 10px auto";
	content[x].style.textAlign = "right";
	x++;
} while (x < len);
  7. We can get interesting features by adding an IF statement within our iteration loop. In this example we are using a MODULUS operator to test for odd or even values, enabling us stripe the rows or columns of our grid. Notice how the IF statement doesn't have an effect unless we iterate the objects 'through' it, unlike CSS, where all we need to do is declare a rule, and all members are effected. 
    8. for (var x = 0, len = rows.length; x < len; x++) {
    if (x % 2 === 0)  {
        rows[x].style.backgroundColor = "#EEE"; 
    }   
}
  8. There are many element manipulation techniques, we'll learn some more in a few weeks, in the meantime, you can see how easy it is to add a class to specific elements using the setAttribute( ) function.
    9. for (x = 0, len = spans.length; x < len; x++) {
    if (x % 2)  {
        spans[x].firstChild.setAttribute("class", "label label-inverse");   
    }   
};
