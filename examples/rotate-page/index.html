about this page

rotate this entire page

the javascript on this page is an attempt to replicate the behavior of the Google search page when you search for "do a barrel roll".

the page relies on the jquery library, which is included in this folder but can be accessed via the cloud by adding this script tag to your webpage.

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.0/jquery.min.js"></script>

The custom jquery code uses the jquery .ready function to ensure the page elements are fully loaded and then adds a 'click' event listener on the #spinner element. This code is a bit more involved than what we worked on in class, but it is able to stop itself when it turns the page one complete revolution.

C and Java programmers might recognize the recursion used in the rotate() function.
