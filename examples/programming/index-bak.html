Javascript Programming and Document Navigation

1. Create an HTML5 webpage with a grid layout system. You can use a pre-existing system such as Twitter Bootstrap or another of your choice.

2. In addition to any other supporting CSS or Javascript files, create your own custom Javascript file and add it to the document using a script tag at the bottom of your page.

3. Create a page layout of at least fifteen blocks of text laid out in a variety of arrangements using CSS classes.

4. In Javascript, use the document.getElementsByClass() function to return values to a variable. The values will be 'references' to the sets of elements.

5. Example Javascript

var spans = document.getElementsByClassName("span3");
var spans = document.getElementsByClassName("span4");
var rows = document.getElementsByClassName("row");

6. Explore the values contained in the variables that you created.
