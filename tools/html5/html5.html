Targeting Markup to user agents

Modernizr
  • Maintain a fine level of control over older browsers with regressive enhancement
  • Dynamically detects HTML5 & CSS3 features features available in the user agent
  • Open source, small footprint Javascript Library
  • use markup to control fallbacks for non-compliant browsers - no need for Javascript.
.no-audio #audio { display: none; /* Don't show Audio */ } <div id="audio"> <audio> <source src="mySong.ogg" /> <source src="mySong.mp3" /> </audio> </div>

Modernizr provides classes to determine user agent capabilities and to style or script fallbacks

Media Queries
  • Tailor display & layout attributes for specific user agent attributes
  • Dynamically detect attributes of user agents in HTML5 or style sheet
  • Style sheet presentations can be tailored to sets of user agents based on device attributes without changing the content
  • Attributes in media queries include 'width', 'height', 'aspect ratio'.
#container { width: 960px; } @media screen and (max-width:999px) { #container { width: 800px; } } @media screen and (max-width:480px) { #container { width: 400px; } }
